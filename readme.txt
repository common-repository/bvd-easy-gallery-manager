One of the easiest gallery plugins available. No complicated uploads or settings to worry about.

== Description ==
With the BVD Easy Gallery Manager you can have beautiful image galleries on your site within minutes. There are no complicated settings required to get your gallery on the page of your choice. Simply upload your photos and put the gallery shortcode where you would like the display the gallery. Uploading photos integrates natively with the Wordpress Media Gallery for ease of use.

For a more custom integration, there are several options available to you for each gallery.

*   Type of Gallery
*   Image Padding
*   Captions
*   Transition Effects
*   Image Size
*   Image Count
*   Auto insert into page

The pro version comes with several more options as well as a Front End photo uploader. The Front-End uploader is optimized for use from a mobile phone. Easily upload photos from your camera or stored photos and instantly make them live. There is a free, 7 day trial of the pro version available from within the plugin so you can see what all the pro version includes before you decide to buy.

This plugin will make requests to an external server to verify if Pro features are available. This is only used for that purpose, we do no collect any other information or use it for any other purpose.

== Installation ==
1. Install the plugin through the Add Plugins directory in your WordPress admin or upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to the BVD Gallery menu option
4. Add a new gallery
5. Place the shortcode where you want the gallery to display following the instructions found at the bottom of the gallery page

== Frequently Asked Questions ==
= Does the plugin contact external servers? =

Yes, the plugin may contact an external licensing server to verify a Pro license.

= What features does the pro version have? =
The pro version comes with a Front-End image uploader, additional gallery types, additional transition effects, infinite image rotation, and pagination options.
